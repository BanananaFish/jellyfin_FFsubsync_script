# Jellyfin FFsubsync Script

## install
```bash
apt install ffmpeg
pip install ffsubsync
```

## dir structure
```
- movies
    - foo movie
        - foo_movie.mp4
        - foo_movie.chi.ass
        - ...
    - bar movie
        - bar_movie.mkv
        - bar_movie.cht.ass
        - ...
    - ...
```
